# TailorFlow – Tailoring Management System

## 📌 Description
A web-based application to manage tailoring shop operations like customers, measurements, orders, and billing.

## 🚀 Features
- Customer management
- Body measurement tracking
- Order lifecycle tracking
- Billing system with invoices
- Dashboard with charts (Chart.js)

## 🛠️ Technologies Used
- PHP 8
- MySQL
- Bootstrap 5
- JavaScript
- Chart.js

## ▶️ How to Run
1. Install XAMPP
2. Place project in htdocs folder
3. Import database (.sql file)
4. Start Apache & MySQL
5. Open in browser: http://localhost/tailoring_new
